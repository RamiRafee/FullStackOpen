# FullStackOpen Solutions

My solutions to the [Full Stack Open](https://fullstackopen.com/en/) course by the University of Helsinki — a free, comprehensive course on modern web development.

## 📚 Course Structure

| Part | Topic |
|------|-------|
| Part 0 | Fundamentals of Web Apps |
| Part 1 | Introduction to React |
| Part 2 | Communicating with Server |
| Part 3 | Programming a server with NodeJS and Express |
| Part 4 | Testing Express servers, user administration |
| Part 5 | Testing React apps |
| Part 6 | Advanced state management |
| Part 7 | React router, custom hooks, styling app with CSS and webpack |
| Part 8 | GraphQL |
| Part 9 | TypeScript |
| Part 10 | React Native |
| Part 11 | CI/CD |
| Part 12 | Containers |
| Part 13 | Using relational databases |

## 🗂️ Repo Structure
```
fullstackopen/
├── part0/
├── part1/
├── part2/
...
└── part13/
```

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript
- **Backend:** Node.js, Express
- **Database:** MongoDB, PostgreSQL
- **Other:** GraphQL, React Native, Docker

## 🚀 Running the exercises

Each part/exercise may have its own setup. Generally:
```bash
npm install
npm start
```
